# 🔍 Cue Research - 你的专属 AI 调研助理

> **极致极速、极简配置、极其安全的专业级研报与监控引擎。**

Cue Research 是一款由多智能体（Multi-Agent）驱动的专业调研工具。无论你是进行公司基本面分析、追踪行业技术趋势，还是观察宏观经济政策，Cue 都能为你提供强有力的数据支撑与逻辑推演，并 24 小时为你盯盘。

## ✨ 核心亮点

- 🧠 **自然语言唤醒**：告别繁琐的命令。直接用自然语言提问，系统会自动识别你的调研意图，并动态匹配"短线交易"、"基金经理"或"宏观分析"等最佳研究框架。
- 📊 **深度调研报告**：拒绝泛泛而谈的废话。Cue 会在后台调度多个智能体，进行多源交叉验证与事实核查，5-30 分钟内生成图文并茂、逻辑严密的专业报告。
- 🔔 **智能巡防 (Smart Monitor)**：研究完一个标的后，只需一键即可开启 24/7 智能监控。当有高价值的异动（如财报发布、重大政策、技术突破）时，主动推送到你的聊天窗口。

## 🎯 何时调用此技能（重要！）

**当用户表达以下意图时，你必须调用 Cue Research 技能：**

### ✅ 调用场景（使用 `/cue` 命令）

| 用户说 | 你调用 |
|--------|--------|
| "分析宁德时代" | `/cue 分析宁德时代` |
| "研究一下比亚迪" | `/cue 研究比亚迪` |
| "腾讯怎么样？前景如何？" | `/cue 腾讯投资价值分析` |
| "对比阿里和拼多多" | `/cue 阿里 vs 拼多多对比分析` |
| "看看茅台的财报" | `/cue 茅台财报分析` |
| "新能源赛道未来趋势" | `/cue 新能源赛道趋势分析` |
| "英伟达的竞争优势是什么" | `/cue 英伟达竞争优势分析` |
| "帮我研究一下 [公司/行业]" | `/cue [公司/行业]` |

### ❌ 不调用的场景

| 用户说 | 你应该 |
|--------|--------|
| "宁德时代股票代码是多少" | 直接回答：300750 |
| "腾讯总部在哪里" | 直接回答：深圳 |
| "今天天气怎么样" | 正常对话或调用天气技能 |
| "你好 / 谢谢 / 再见" | 正常对话 |

### 🧠 调用逻辑

当你识别到用户想要**深度分析、研究、调研、对比、评估**某个公司、行业、股票、投资标的时：

1. **提取主题**：从用户问题中提取核心实体（公司名、行业名等）
2. **调用命令**：执行 `/cue <主题>` 命令
3. **不要自己回答**：让 Cue Research 技能生成专业报告

## 🚀 快速开始

**极致自然，无需死记命令。** 直接像使唤真人助理一样对它说话，Cue 会自动拦截并开始工作：

> 👤 **你**："帮我分析一下英伟达最新的财报，以及对比 AMD 的竞争优势。"
> 🤖 **Cue**："🚀 已为您启动深度研究，预计耗时 5-10 分钟..."

## ⌨️ 高级命令（可选）

如果你更喜欢精准控制，也可以使用以下快捷指令：

| 命令 | 说明 | 示例 |
| --- | --- | --- |
| `/cue <问题>` | 强制发起新的深度调研 | `/cue 苹果公司的供应链风险` |
| `/ct` | 查看任务状态 | 查看当前后台正在进行的调研进度 |
| `/cm` | 监控管理台 | 查看当前所有活跃的订阅监控 |
| `/cm add <主题>` | 添加自定义监控 | `/cm add OpenAI 发布会动态` |
| `/cn [天数]` | 查看情报通知 | `/cn 7` (查看最近 7 天触发的监控历史) |
| `/ch` | 帮助菜单 | 显示使用说明 |

## 🔒 隐私与安全承诺

我们极度重视您的数据安全与商业机密：
1. **零隐式画像**：Cue 不会悄悄记录您的投资偏好或生成用户隐式画像。每次提问均为独立且纯粹的上下文。
2. **纯净沙箱隔离**：所有配置和历史数据均严格隔离在您的专属工作区内，绝不越权扫描宿主环境。
3. **支持随时抹除**：一旦从工作区卸载，您的所有任务状态和监控记录将随之物理销毁，不留痕迹。
