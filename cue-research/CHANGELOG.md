# 📋 更新日志 (Changelog)

## [1.0.0] - 2026-03-10

### 🎉 Cue Research 正式发布

#### 品牌升级
- 🏷️ **更名为 Cue Research**：更精准定位专业调研能力，提升市场辨识度
- 📄 **新增 SKILL.md**：完整的技能说明文档，支持自然语言唤醒
- ✨ **自然语言交互**：无需死记命令，像使唤真人助理一样对话即可

### 🎉 核心功能发布

#### 核心功能
- ✨ **深度调研引擎**：基于 CueCue API 的多智能体异步流式研究，5-30 分钟生成专业报告
- 🧠 **动态 Prompt 引擎**：自动识别研究意图（宏观/短线/财报/产业），匹配最佳分析框架
- 🔔 **智能监控系统**：7x24 小时 Cron 守护进程，支持 11 种预定义监控模板
- 📊 **交互式卡片 UI**：进度条实时展示、报告完成通知、一键开启推荐监控

#### 技术架构
- 🚀 **纯异步 I/O**：0 子进程设计，完美适配 OpenClaw 高并发场景
- 🛡️ **安全隔离层**：用户级工作空间隔离，原子化 JSON 写入防损坏
- ⚡ **性能优化**：搜索超时保护 (10s)、研究总超时 (15min)、防雪崩并发锁
- 🎯 **智能触发器**：纯 NLP 关键词匹配，40% 命中率阈值，无大模型依赖
- 📦 **独立通知存储**：监控触发记录单独存管，支持历史查询与详情链接
- 🧠 **NLU 意图识别**：基于打分机制的自然语言意图检测，支持免命令唤醒
