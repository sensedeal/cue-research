# 🔒 隐私与数据处理声明 (Privacy Policy)

我们非常重视您的隐私。Cue 作为一个透明、安全的 OpenClaw 技能插件，严格遵循最小化权限原则。

## 1. 我们收集和处理哪些数据？
* **您的输入问题**：您通过 `/cue` 或 `/cm` 命令输入的调研主题。
* **平台标识**：仅使用脱敏的平台 User ID（如 `feishu-ou_xxx`）作为数据隔离文件夹的名称。

## 2. 数据会发送到哪里？
为了提供强大的分析能力，您的**当前调研问题**会被临时发送至以下受信任的第三方 API：
* **CueCue API (`cuecue.cn`)**：用于多智能体深度分析推理。
* **Tavily API (`api.tavily.com`)**：用于全网实时新闻和数据检索。

## 3. 我们不做什么？（v1.0 安全承诺）
* 🚫 **不收集用户画像**：我们**不再**记录您的投资偏好、历史提问轨迹或对您进行隐式画像。每次提问均为独立上下文。
* 🚫 **不读取宿主环境**：插件被严格限制在 OpenClaw 分配的 `.cuecue` 用户隔离目录下读写，绝不触碰系统的其他文件或密钥。
* 🚫 **不利用数据进行训练**：我们的服务条款禁止使用用户的私域调研问题用于大模型预训练。

## 4. 数据清理
您可以随时要求管理员清空 OpenClaw `workspaces` 目录下的个人文件夹，您的所有任务状态和监控配置将随之彻底抹除，不留任何痕迹。
