## 🧪 测试场景

### 场景 1: 完整研究流程（15-30 分钟）

**步骤**:
1. 在 Feishu 中发送："分析宁德时代"
2. 等待启动通知（立即）
3. 等待进度通知（每 5 分钟）
4. 等待完成通知（5-30 分钟后）

**预期结果**:
```
🚀 启动通知（立即收到）
   - 主题：分析宁德时代
   - 视角：产业研究视角
   - 链接：https://cuecue.cn/c/xxx

🔔 进度通知（每 5 分钟）
   - 已用时：5 分钟
   - 当前阶段：收集行业数据
   - 预计剩余：25 分钟

✅ 完成通知（研究完成后）
   - 核心摘要
   - 报告链接
   - 推荐监控
   - 操作按钮
```

**记录**:
- [ ] 启动通知收到时间：____
- [ ] 第一次进度通知时间：____
- [ ] 完成通知收到时间：____
- [ ] 总耗时：____分钟

### 场景 2: 快捷回复交互（5 分钟）

**前置条件**: 有一个已完成的研究

**步骤**:
1. 发送："状态"
2. 发送："创建监控"
3. 发送："Y"
4. 发送："追问"
5. 发送："取消"

**预期结果**:
```
"状态" → 📊 研究任务列表
"创建监控" → ✅ 已为您创建监控
"Y" → ✅ 已为您创建监控
"追问" → 💬 追问问题
"取消" → ✅ 已取消任务
```

**记录**:
- [ ] 状态查询：✅/❌
- [ ] 创建监控：✅/❌
- [ ] 追问问题：✅/❌
- [ ] 取消任务：✅/❌

### 场景 3: 监控功能（10 分钟）

**步骤**:

**预期结果**:
```
"/cm add" → ✅ 成功添加监控
"/cm" → 🔔 监控列表
"/cn 7" → 通知历史记录
```

**记录**:
- [ ] 创建监控：✅/❌
- [ ] 查看列表：✅/❌
- [ ] 通知历史：✅/❌

### 场景 4: NLU 识别测试（5 分钟）
